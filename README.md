# "Codespace Cloud" devcontainer

`mcr.microsoft.com/devcontainers/base:debian` configured to run [Supervisor](https://supervisord.org/) on start.

Intended to be used with [ansible-csc](https://github.com/meskuk/ansible-csc) and [cscloud](https://github.com/meskuk/cscloud).

When running this devcontainer, please ensure to _remain connected_ for the first start, so the Ansible playbooks can run!
